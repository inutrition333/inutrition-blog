# 老闆快速發文備忘單

一張紙就能看完。遇到問題先對照這張，真的都不行再找人。

---

## 日常發文（最簡單的路）

1. 打開 👉 **https://inutrition-blog.vercel.app/write**
2. 輸入管理員密碼 → 進發文頁
3. 填四個欄位：**標題**、**分類**、**摘要**、**文章內文（貼整段 markdown）**
4. 按「送出發文」→ 看到綠色「已送出 ✅」訊息就完成
5. 約 1 分鐘後文章會自動出現在部落格

> 其他欄位（發布日期、閱讀時間、campaign、檔名、主圖）都有合理預設值，不用管。需要的話點「顯示進階欄位」調整。

## 如果出錯了（Top Down 自救 SOP）

### 🟡 送出後看到紅色錯誤訊息

把紅色訊息**整句讀完**，通常已經告訴你該怎麼做：

| 錯誤訊息 | 怎麼辦 |
|---|---|
| `密碼錯誤` | 輸入錯了，重新輸入 |
| `還有必填沒填：xxx` | 把 xxx 欄位補滿 |
| `檔名 xxx.md 已經存在` | 點「進階欄位」改一個不同的檔名（slug） |
| `連線失敗` | 網路問題，等一下再試 |
| 其他錯誤 | 截圖 → 照下面的步驟 |

### 🟡 整個頁面打不開 / 白屏 / 按鈕沒反應

1. 強制重整：**Mac 按 `Cmd + Shift + R`**，Windows 按 `Ctrl + Shift + R`
2. 換一個瀏覽器試試（Chrome / Safari / Edge 都可以）
3. 都不行 → 往下看

### 🟡 `/write` 完全壞掉，但文章很急

**備援路徑**：改用完整後台
- 打開 👉 **https://inutrition-blog.vercel.app/admin/**
- 用 GitHub 帳號登入（如果第一次使用，要先請系統管理員把你加成協作者）
- 點左邊「部落格文章」→「New 部落格文章」→ 手動填欄位、貼內文 → 儲存

## 還是不行？兩個外援

### 外援 1：問你自己的 Claude

截圖錯誤訊息 + 說明你在做什麼，直接貼給 Claude。大部分時候 Claude 能告訴你怎麼解，或至少判斷「是不是你能自己處理」。

**丟給 Claude 的範本：**
> 我在 inutrition-blog.vercel.app/write 發文章時遇到這個錯誤（附截圖）。我已經試過：[強制重整 / 換瀏覽器 / 其他]。幫我看該怎麼辦？

### 外援 2：聯絡系統管理員

- 敏恩 Amber：（手機 / email 填這裡）
- 如果敏恩不在且文章非常急：先把整篇 markdown 原稿寄到上述 email，之後回來補發就好。**不要因為急硬打入損壞的系統**，可能變成重複發文或髒資料。

## 寫作流程建議（給用 Claude 產稿的人）

1. 在自己的 Claude 下指令：「用愛營養部落格的風格，寫一篇關於 XXX 的文章」
2. 得到 markdown 草稿後，複製**整段內文**（不包含上面的 `---` frontmatter 區塊，那個 /write 頁面會自動幫你組）
3. 到 /write 頁面貼到「文章內文」欄位，填好標題/分類/摘要 → 送出

markdown 裡可以用的特殊區塊：
- `:::product` — 產品導購卡片
- `:::sunny` — Sunny 小叮嚀
- `:::related` — 延伸閱讀

詳細語法在「給使用者的操作手冊.md」裡有。
